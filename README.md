# Task 4 - Responsible AI & Model Interpretation

## Objective

To understand Responsible AI concepts such as fairness, bias, transparency, and model interpretability using a Machine Learning model.

## Tools Used

- Python
- Scikit-learn
- Pandas
- Matplotlib

## Dataset

Iris Dataset

## Description

In this project, a Machine Learning model was trained using the Iris dataset. Feature importance analysis was performed to understand which features influenced the model predictions. Responsible AI principles such as fairness, bias awareness, and data privacy were also discussed.
